Energy Adaptive Routing Protocol (EARP) in WSN
EARP is a modification of SPIN protocol which is used in WSN for secure communication. To make the security protocol energy efficient, some additional functions were added which include progressive route request, backing route error, pseudo-random delay, critical energy state routing and overheard route cache maintainer
